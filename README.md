# CodeAlpha Network Sniffer

## Project Overview
This project is a simple network packet sniffer developed in Python using the Scapy library. It captures network packets, displays information about each packet, and saves the captured packets to a Wireshark-compatible `.pcap` file for further analysis.

## Features
- Captures live network packets
- Displays source MAC and Destination MAC addresses
- Displays source IP and destination IP addresses (IPv4 & IPv6)
- Displays the protocol and packet length
- Displays Source port and destination port numbers
- Displays packet payload (if available)
- Saves captured packets to `captured_packets.pcap`
- Compatible with Wireshark for packet analysis

## Technologies Used
- Python 3
- Scapy
- Npcap
- Wireshark

## How to Run
1. Install Python 3.
2. Install Scapy:
   ```bash
   pip install scapy
   ```
3. Install Npcap.
4. Run the program:
   ```bash
   python network_sniffer.py
   ```
5. Generate network traffic (for example, by opening websites or using network applications).
6. Stop the program using **Ctrl + C**.
7. Open `captured_packets.pcap` in Wireshark to analyze the captured packets.

## Project Files
- `network_sniffer.py` – Python source code for the network sniffer.
- `captured_packets.pcap` – Sample packet capture generated by the program.
- `README.md` – Project documentation.
